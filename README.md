# nurulazamDev Portfolio v_3

This is the repository for the latest version of the personal portfolio website of Mohammad Nurul Azam, a MERN Stack Developer.

---

## 📚 Installation & Setup

1. Clone the repository
   ```bash
   git clone https://github.com/nurulazamDev/portfolio-v2.1.0.git
   cd portfolio-v2.1.0
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Set up environment variables
   - Rename `.env.local.example` to `.env.local` and update the values as needed.
4. Run the development server
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:3000`
